## 🐄 Sobre mí

Trabajo en la intersección entre **Ciencia de Datos, IA y ganadería** para transformar
la selección genética bovina. Creo que el futuro del campo está en los datos.

- 🔭 Trabajando en: modelos predictivos para mejoramiento genético bovino
- 🧬 Explorando: genómica, EPDs, índices de selección y BLUP/GBLUP
- 🌱 Aprendiendo: ML aplicado a datos fenotípicos y genotípicos
- 🤝 Busco colaborar con: ganaderos, investigadores y agtech startups
- 💬 Pregúntame sobre: Python para agronomía, análisis de pedigríes, EBVs
- ⚡ Fun fact: un buen modelo genético puede valer más que el mejor toro
